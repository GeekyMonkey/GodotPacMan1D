# **Godot PacMan 1D**

A clone of the fantastic [Paku Paku](https://abagames.github.io/crisp-game-lib-11-games/?pakupaku) browser game as an example for the kids at CoderDojo Ennis.  Written in the GdScript language.

Features:

* Simple one key (tap) control. With a keyboard or mouse you could alternatively use left/right a/d  or mouseL/mouseR to control the direction.
* Integer scaling. The pixels are rendered at the same relative size at any scaling size, without any filtering.
* 16/16 sprites based on the original PacMan arcade game
* Sounds from the origina PacMan arcade game
